# movie_EDA
###### 패스트캠퍼스 프로젝트1
###### pandas를 활용한 영화제 수상작 및 우리나라 boxoffice 흥행작 EDA


### project 설명
세계적인 영화제 수상작과 우리나라 boxoffice 흥행작 Top10의 특징을 분석, 그 상관관계를 비교하여 시각화
